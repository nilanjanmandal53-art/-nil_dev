Digital Compass Wallet:
​A sleek, modern, and responsive Digital Wallet UI built using HTML5, CSS3, and JavaScript. This project focuses on a "Compass" navigation theme, blending traditional navigation concepts with modern financial technology.
​📌 Project Overview
​As part of my journey in CSC (AI/ML), I developed this project to master the art of creating intuitive user interfaces. The Digital Compass Wallet features a clean dashboard where users can track their balance, view recent transactions, and navigate through their financial "directions."
​🚀 Key Features
​Interactive UI: A compass-inspired navigation menu for a unique user experience.
​Responsive Design: Fully optimized for Mobile, Tablet, and Desktop screens using CSS Flexbox/Grid.
​Dynamic Wallet Balance: JavaScript logic to handle real-time UI updates (simulated).
​Transaction History: A styled list showing recent activity with "Inflow" and "Outflow" color-coding.
​Glassmorphism: Modern UI effects using backdrop-filter and soft shadows.
​🛠️ Tech Stack
​Structure: HTML5 (Semantic tags for SEO)
​Styling: CSS3 (Custom Variables, Keyframe Animations)
​Functionality: Vanilla JavaScript (ES6+ features)
